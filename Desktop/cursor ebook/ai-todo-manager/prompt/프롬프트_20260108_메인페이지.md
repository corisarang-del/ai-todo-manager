# 프롬프트 - 메인 페이지 생성

**작성 시각**: 2026-01-08 17:43:15

## 요청 내용

PRD.md 문서를 기반으로 `app/page.tsx` 메인 화면을 구성

### 주요 기능
1. 할 일 목록 표시
2. 할 일 관리 (추가, 수정, 삭제, 조회)
3. 검색, 필터, 정렬
4. 사용자 정보 표시
5. 로그아웃 기능

### 컴포넌트 구성
1. **Header**: 서비스 로고/이름, 로그인 사용자 정보, 로그아웃 버튼
2. **Toolbar**: 검색창, 상태 필터 (완료/미완료), 우선순위 필터, 정렬 기준 드롭다운
3. **Main Area**: 
   - 좌측/상단: TodoForm 컴포넌트
   - 우측/하단: TodoList 컴포넌트 (TodoCard로 구성)

### 구현 지침
- `components/todo`의 TodoForm, TodoList, TodoCard 컴포넌트 재사용
- Shadcn/ui 컴포넌트 활용
- Mock 데이터로 화면 구조 구성 (Supabase 연동 전)
- Flex/Grid를 활용한 반응형 레이아웃
