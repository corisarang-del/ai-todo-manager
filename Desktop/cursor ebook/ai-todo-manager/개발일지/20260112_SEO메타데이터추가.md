# SEO 메타데이터 추가

**작성시각**: 2026-01-12 15:21

## 해결하고자 한 문제

Next.js 프로젝트의 `app/layout.tsx`에 검색 엔진 최적화(SEO)와 소셜 미디어 공유를 위한 메타데이터가 부족했음.

## 구현 내용

### 1. 기본 메타데이터 추가
- **title**: "AI 할 일 관리 서비스"
- **description**: "AI가 도와주는 똑똑한 할 일 관리 서비스"
- **keywords**: 할 일 관리, TODO, AI, 생산성, 일정 관리, task management
- **authors, creator, publisher**: AI Todo Manager

### 2. 검색 엔진 최적화
```typescript
robots: {
  index: true,
  follow: true,
  googleBot: {
    index: true,
    follow: true,
  },
}
```

### 3. Open Graph 태그 (Facebook, LinkedIn 등)
- type: "website"
- locale: "ko_KR"
- url, title, description, siteName 설정

### 4. Twitter Card 태그
- card: "summary_large_image"
- title, description, creator 설정

### 5. Viewport 설정
- width: "device-width"
- initialScale: 1
- maximumScale: 1

### 6. 기타 개선사항
- HTML lang 속성을 "en"에서 "ko"로 변경
- Google Search Console verification 필드 추가

## 해결된 것

✅ SEO 메타데이터 완전히 추가됨
✅ 소셜 미디어 공유 시 올바른 정보 표시 가능
✅ 검색 엔진 크롤링 최적화
✅ 한국어 콘텐츠 명시

## 해결되지 않은 것

- Google Search Console verification code는 실제 배포 시 업데이트 필요
- Open Graph 이미지는 추후 추가 필요 (현재 이미지 없음)

## 향후 개발을 위한 컨텍스트

1. **Open Graph 이미지 추가**: 소셜 미디어 공유 시 표시될 이미지 생성 및 추가 필요
2. **동적 메타데이터**: 페이지별로 다른 메타데이터가 필요한 경우 각 page.tsx에서 개별 설정 가능
3. **Favicon 추가**: app/favicon.ico 또는 app/icon.png 추가 권장
4. **Sitemap 생성**: app/sitemap.ts 파일로 자동 sitemap 생성 고려

## 참고사항

Next.js 16.1.1 버전의 Metadata API를 사용하여 타입 안전성 확보
